For add my SimpleMenu API you need to add in pom.xml:

     <repository>
            <id>jitpack.io</id>
            <url>https://jitpack.io</url>
        </repository>

        <dependency>
            <groupId>com.github.Falzik</groupId>
            <artifactId>SimpleMenu</artifactId>
            <version>v1.0</version>
        </dependency>
